jQuery Custom Gantt API

대/중/소 분류 일정 데이터를 렌더링하는 jQuery 플러그인의 properties, methods, static methods, events, data schema 레퍼런스입니다.

$('#chart').customGantt({
  data: ganttData,
  viewMode: 'week',
  initialExpandLevel: 'medium'
});

Properties

data

Arraydefault: []

계층화된 Gantt 일정 데이터입니다. 분리 배열을 사용하는 경우 $.customGantt.buildHierarchy()로 먼저 변환합니다.

title

Stringdefault: "Project Gantt"

차트 toolbar에 표시할 제목입니다.

startDate

String | nulldefault: null

차트 표시 범위의 시작일입니다. 생략하면 전체 데이터의 최소 start 값을 자동으로 사용합니다.

endDate

String | nulldefault: null

차트 표시 범위의 종료일입니다. 생략하면 전체 데이터의 최대 end 값을 자동으로 사용합니다.

viewMode

Stringdefault: "day"

날짜 표시 단위입니다. day, week, month를 지원합니다.

$('#chart').customGantt({ viewMode: 'month' });

initialCenterDate

String | Date | nulldefault: null

초기 렌더링 직후 이동할 위치입니다. YYYY-MM-DD, Date, today, start, end를 지원합니다.

initialExpandLevel

String | nulldefault: null

초기 렌더링 시 표시할 분류 깊이입니다. large, medium, small을 지원합니다. 생략하면 initialCollapsed 값을 기준으로 결정합니다.

excludeWeekends

Booleandefault: false

day view에서 토요일과 일요일 컬럼을 숨깁니다. 일정 종료일이 주말이면 bar는 해당 주 금요일까지 표시됩니다.

colorRenderer

Function | nulldefault: null

원본 객체를 분석해서 bar 색상을 반환합니다. CSS 색상명, hex, rgb 문자열을 지원하며 데이터의 color보다 우선 적용됩니다.

colorRenderer: function (item) {
  if (item.status === 'FINISHED') return 'green';
  if (item.status === 'READY') return '#f97316';
  return 'rgb(75, 115, 233)';
}

barLabelRenderer

Function | nulldefault: null

bar 내부에 표시할 문자열을 반환합니다. null 또는 undefined를 반환하면 기본값인 일정명 진행률%를 표시합니다.

barLabelRenderer: function (item, context) {
  return item.status + ' · ' + context.progress + '%';
}

Additional properties

dayWidthweekWidthmonthWidthrowHeightlocaleshowTodayinitialCollapsedcolorThemeignoreDataColors

셀 크기, locale, 오늘 표시선, 이전 버전 호환용 초기 접힘 옵션, 색상 테마와 데이터 색상 우선순위를 제어합니다. 기본값은 README 옵션 표를 참고합니다.

Methods

expandAll()

대분류와 중분류를 모두 펼쳐 소분류 작업까지 표시합니다.

$('#chart').customGantt('expandAll');

collapseAll()

대분류와 중분류를 모두 접습니다.

$('#chart').customGantt('collapseAll');

expandToLevel(level)

level: "large" | "medium" | "small"

지정한 분류 깊이까지만 펼칩니다.

$('#chart').customGantt('expandToLevel', 'medium');

scrollToDate(date, align)

date: Datealign: "start" | "center" | "end"

차트를 지정한 날짜 위치로 스크롤합니다.

$('#chart').customGantt('scrollToDate', new Date(2026, 4, 1), 'center');

update(options)

플러그인 호출에 options 객체를 다시 전달하면 설정을 병합하고 차트를 다시 렌더링합니다.

$('#chart').customGantt({ viewMode: 'week' });

destroy()

이벤트, tooltip, context menu, modal과 차트 DOM을 정리합니다.

$('#chart').customGantt('destroy');

Static Methods

$.customGantt.buildHierarchy(largeList, mediumList, smallList, options)

분리된 대분류, 중분류, 소분류 배열을 차트가 사용하는 hierarchy 배열로 변환합니다. 네 번째 인자로 필드명 매핑을 전달할 수 있습니다.

var ganttData = $.customGantt.buildHierarchy(
  response.largeList,
  response.mediumList,
  response.smallList
);

Events

현재 버전은 외부에서 구독할 수 있는 custom event를 제공하지 않습니다. 클릭, collapse, context menu 이벤트는 플러그인 내부에서 처리합니다.

Field Type Scope Description

Validation

Invalid date range

start가 end보다 늦으면 해당 row는 차트 범위와 요약 계산에서 제외됩니다. bar 대신 오류 배지가 표시되며, 배지를 우클릭하면 원본 기간과 상세 정보를 확인할 수 있습니다.

일치하는 API 항목이 없습니다.
